### Summary (max 154 chars)

### Description

**Volume & Video Master 1000%** gives you professional audio controls for any video or audio playing in Firefox.

### Features

🎚️ **Volume Boost** — Increase volume beyond the browser's 100% limit, up to 1000%. Perfect for quiet videos or laptops with weak speakers.

⏱️ **Playback Speed** — Adjust speed from 0.25x to 5x. Great for speeding through lectures or slowing down tutorials.

🔊 **Bass Boost** — Enhance low frequencies for richer, fuller audio.

🗣️ **Voice Boost** — Clarify speech frequencies for podcasts, lectures, and dialogue-heavy content.

🎧 **Mono Audio** — Mix stereo channels to mono for single-earbud listening or hearing-impaired accessibility.

### Smart Settings

- **Global Mode:** Apply the same settings to every website
- **Per-Site Mode:** Save different settings for YouTube, Netflix, Spotify, etc.
- **Disabled Mode:** Turn off the extension for specific sites
- **Persistent:** Your preferences are saved across browser sessions

### Works Everywhere

The extension automatically detects video and audio elements on any website — YouTube, Netflix, Twitch, Spotify, news sites, online courses, and more. No configuration needed.

### Privacy First

- No data collection or tracking
- No external servers — all processing happens locally
- No ads or affiliate links
- Open source

## Important Firefox-Specific Notes

### Source Code Submission
**Firefox REQUIRES source code for extensions built with build tools** (WXT, Webpack, Vite, etc.). You must upload the sources.zip file generated by `pnpm zip:firefox`.

### Build Instructions for Reviewers
Include these instructions in the review notes:

```
Build instructions:
1. Extract source archive
2. Install dependencies: pnpm install
3. Build: pnpm build:firefox
4. Output: .output/firefox-mv2/

Requirements:
- Node.js 18+
- pnpm
```

### Permissions Explanation
Firefox will ask about these permissions:

**`<all_urls>`** — Required to access video/audio elements on all websites. Without this, the extension would only work on a whitelist.

**`tabs`** — Required to send updated settings to tabs matching the current hostname.

**`storage`** — Required to save user preferences locally.

### data_collection_permissions
This extension has `data_collection_permissions` set to all `false` in the manifest. We do not collect any user data.

## Common Firefox Review Issues

### Issue: "Your add-on uses a custom CSP"
**Not applicable** — We don't set a custom CSP.

### Issue: "Your add-on uses eval() or similar"
**Not applicable** — No eval, new Function(), or setTimeout with strings.

### Issue: "Your add-on loads remote code"
**Not applicable** — All code is bundled. No external scripts.

### Issue: "Your add-on has excessive permissions"
**Mitigation:** The `<all_urls>` permission is justified because:
1. The extension processes HTML5 media elements
2. Media elements exist on virtually every website
3. A whitelist approach would be impractical and user-hostile
4. The extension only touches `<video>` and `<audio>` elements
5. No page content, forms, or personal data is accessed
